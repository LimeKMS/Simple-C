Simple HashTable by Liam Rigney N01450656
9/12/22

Purpose: To have a program that can read a .txt file and store the information in it
to a hash table with no GUI

Internal Files: main.c (Main program)
External Files: File.txt (input), text.txt (Documentation/Instructions)

How to run program:
Simply input these lines of code into your C compiler while main.c and File.txt
are both in your directory

gcc main.c -o main
./main File.txt
